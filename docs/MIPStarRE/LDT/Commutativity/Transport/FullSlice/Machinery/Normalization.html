    Full-slice normalization and self-consistency machinery #

    Normalization-condition bounds, evaluated projective submeasurements, and full-slice and evaluated-slice self-consistency estimates used by the scalar-to-tensor bridge chain.

    Ex-private definitions are tensor-form machinery per architecture decision #713; downstream code should use the scalar public API exposed by the full-slice transport theorems.

    References #

    • arXiv:2009.12982, Section 11.

    Paper lem:normalization-condition (commutativity-G.tex line 309).

    For a sub-measurement P and projective sub-measurement Q, the sandwiched family C_{a,b} = Q_b · P_a · Q_b satisfies the closenessOfIP normalization condition ∑_a (∑_b C_{a,b}) (∑_b C_{a,b})ᴴ ≤ I.
