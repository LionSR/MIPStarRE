    Overlap-only version of the first stability estimate.

    This is not the paper's boundedness-driven scalar proof of clm:g-comm-stability: it bounds the current SDD package through the slice SSC overlap term ⟨ψ,(I-G^y)⊗G^y ψ⟩. It remains useful as an internal overlap lemma while the scalar-chain API is being completed.
