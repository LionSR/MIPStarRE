    Product rules for finite-support distribution averages #

    This module contains product and marginalization rules for operator-valued uniform averages. The scalar product rules, together with the common PMF-weighted finite-sum identities on which both scalar and operator rules depend, are in MIPStarRE.LDT.Basic.DistributionUniformSums and MIPStarRE.LDT.Basic.PMFAverages.
