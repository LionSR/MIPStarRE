    Section 9 — Saturated canonical SDP witnesses #

    This module contains the zero-slack variant of the canonical SDP output used in the self-improvement argument. The canonical block SDP supplies a feasible matrix X; when its slack diagonal block is zero, the extracted polynomial blocks form a complete measurement without using the auxiliary dominance condition (I \le Z).

    MatrixSdpCanonicalBlockIndex params → MakingMeasurementsProjective.MatrixOperator model.space

    Move the canonical primal slack block into the distinguished polynomial block.

    Paper origin: references/ldt-paper/self_improvement.tex:177-190. The paper passes from an optimal canonical block matrix to a saturated paper primal measurement. This block family implements the source-faithful completion step: the none block is set to zero, and its positive mass is added to the fixed polynomial block sdpDistinguishedPolynomial params. This avoids the auxiliary route which proves saturation from an additional bound I ≤ Z.
