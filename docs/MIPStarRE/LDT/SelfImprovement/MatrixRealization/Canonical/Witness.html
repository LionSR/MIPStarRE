    Section 9 — Canonical matrix SDP witnesses #

    This module contains the optimal-witness packages extracted from the canonical matrix SDP. The preceding canonical module proves the block-diagonal algebra, including the dual slack identities and saturation of the slack block under the retained dominance hypothesis (I \le Z). This file records the paper-form optimal pair, the dominance-carrying successor package, and the measurement witnesses used by the self-improvement bridge.
