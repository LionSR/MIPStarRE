    Section 9 -- Canonical SDP strong duality #

    This compatibility module preserves the historical import path for the canonical finite-dimensional matrix SDP strong-duality development. The feasibility and compactness preliminaries live in Canonical.StrongDuality.Basic; the separation and zero-gap argument lives in Canonical.StrongDuality.Separation. The short objective-bound lemmas below remain here because they are useful consequences of the same canonical SDP interface.
