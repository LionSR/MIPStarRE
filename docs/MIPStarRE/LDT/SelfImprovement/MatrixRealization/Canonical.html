    Section 9 — Canonical matrix SDP dual and slackness layer #

    This module contains the canonical objective and dual operators, the dual slack block identities, and the slack-block saturation step used to extract the paper-form primal normalization. The optimal-witness packages built from these canonical facts live in MatrixRealization/Canonical/Witness.lean.
