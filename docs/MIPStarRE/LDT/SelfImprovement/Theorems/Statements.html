    Prop

    A reduced SDP witness for the currently formalized self-improvement pipeline.

    The paper's lem:sdp eventually supplies strong duality, complementary slackness, and a concrete matrix-level optimal witness. The current Lean pipeline only consumes the weaker facts recorded here: the primal witness is a full measurement (T.total = 1), the dual witness is PSD, and it dominates every averaged point operator.

    • primalTotalOperator : T.total = 1
    • dualPositive : 0 ≤ Z
    • dualFeasible (g : Polynomial params) : 0 ≤ sdpDualSlackOperator params strategy Z g
