    strategy.classicalRoleSymmStrategy.selfConsistencyFailureProbability ≤ delta

    The role-register symmetrized strategy's self-consistency is bounded by any available cross-prover point-agreement bound.

    The full low-individual-degree failure surrogate does not itself provide such a point-agreement bound: its self-consistency branch contains the separate SSC defects of the two point measurements. This conditional lemma records the correct bridge when an independent point-agreement estimate is available.
