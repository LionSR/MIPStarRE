    Ordinary restricted-slice recursion #

    Compatibility module for the ordinary x-restricted successor route in the mainFormal construction. It re-exports the successor weighted bounds, the restricted-probability package, and the recursive-slice target (MainFormalSuccessorRecursiveSlices).

    The former slice-data witness package, which supplied concrete restricted same-space strategies as an additional input, has been removed. In the active successor route the corresponding restricted-slice data are constructed inside MainInductionStep.mainInduction, not supplied as a separate hypothesis of a source-facing theorem.
