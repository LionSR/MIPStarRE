    Two-Space Projective Strategies: Same-Space Projection Lemmas #

    This module contains the same-space-to-two-space projection lemmas.

    Projection lemmas keep the same-space-to-two-space embedding transparent. They are deliberately definitional: SameSpaceProjStrat extends the general paper-faithful ProjStrat, so Lean's generated toProjStrat parent accessor is the canonical forgetful map.

    Source-level alias for Lean's generated toProjStrat parent accessor.

    The actual parent projection comes from the extends ProjStrat params ι ι clause on SameSpaceProjStrat; this alias gives blueprint/checkdecl tooling a named source declaration for the same forgetful map.
