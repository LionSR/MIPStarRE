    Section 12 — Standing pasting context: wrapper restatements #

    Thin wrappers that restate thm:ld-pasting, lem:ld-pasting-sub-measurement, and the downstream pasting bridge lemmas using a single LdPastingContext argument instead of the un-bundled Section-12 hypothesis list.

    These wrappers introduce no new proof content: each body is a thin projection-and-apply wrapper around the corresponding un-bundled statement.
