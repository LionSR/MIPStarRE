    Section 12 — Standing pasting context #

    First-class Lean bundle for the "standing pasting context" of thm:ld-pasting (def:ld-pasting-context in the blueprint).

    The context packages exactly the data and hypotheses that thm:ld-pasting, lem:ld-pasting-sub-measurement, and the downstream pasting bridge lemmas share: an (ε,δ,γ)-good symmetric strategy for the (m+1,q,d) low individual degree test, a slice-indexed polynomial family satisfying the four pasting input properties, and an integer k ≥ 400md.

    This file only introduces the carrier and re-exposes the paper's standing ν / σ abbreviations on top of it. It performs no proof work: all downstream theorems are re-stated via thin wrappers in later modules.

    Type (max u_1 u_2)

    The standing pasting context of thm:ld-pasting.

    This bundle records, for a fixed outcome type ι and fixed ambient parameters params (interpreted as the slice dimension m), all of the data and hypotheses threaded implicitly through thm:ld-pasting, its sub-measurement variant lem:ld-pasting-sub-measurement, and the Section 12 pasting bridge lemmas:

    • an (ε,δ,γ)-good symmetric strategy at dimension m+1;
    • a slice-indexed projective submeasurement family {G^x}_{x ∈ 𝔽_q} in polysub{m}{q}{d} together with completeness κ, point-consistency ζ, strong self-consistency, and the averaged boundedness input;
    • the source-side low-degree inequality d ≤ q and strict positivity 0 < d used in the downstream sandwich and H-consistency calculations;
    • the pasting iteration count k ≥ 400md, including the 1 ≤ k positivity needed by the Bernoulli tail recurrence.

    The derived error abbreviations ν and σ from the paper are provided via LdPastingContext.nu / LdPastingContext.sigma.

    • strategy : SymStrat params.next ι

      The (ε,δ,γ)-good symmetric strategy at dimension m+1.

    • eps : Error

      Axis-parallel-line error parameter ε.

    • delta : Error

      Point self-consistency error parameter δ.

    • gamma : Error

      Diagonal-line error parameter γ.

    • kappa : Error

      Averaged completeness parameter κ.

    • zeta : Error

      Averaged self-improvement / pasting interface parameter ζ.

    • good : self.strategy.IsGood self.eps self.delta self.gamma

      (ε,δ,γ)-goodness of the symmetric strategy.

    • gamma_le_one : self.gamma ≤ 1

      Small-parameter hypothesis γ ≤ 1 used throughout Section 12.

    • zeta_le_one : self.zeta ≤ 1

      Small-parameter hypothesis ζ ≤ 1 used throughout Section 12.

    • dq_le_q : params.d ≤ params.q

      Source-style low-degree inequality d ≤ q.

    • d_pos : 0 < params.d

      Strict positivity 0 < d required by the sandwich lemmas.

    • family : IdxPolyFamily params ι

      The slice-indexed polynomial family {G^x}_{x ∈ 𝔽_q}.

    • complete : self.family.Complete self.strategy.state self.kappa

      Averaged completeness of the slice family (item:ld-pasting-completeness).

    • consistent : self.family.ConsistentWithPoints self.strategy self.zeta

      Averaged point-consistency (item:ld-pasting-consistency).

    • selfConsistent : self.family.StronglySelfConsistent self.strategy.state self.zeta

      Averaged strong self-consistency (item:ld-pasting-self-consistency).

    • bounded : IdxPolyFamily.SliceBoundednessInput self.strategy self.family self.zeta

      Averaged boundedness input (item:ld-pasting-boundedness).

    • k : ℕ

      Pasting iteration count k.

    • hk_pos : 1 ≤ self.k

      Positivity of the iteration count.

    • hk : 400 * params.m * params.d ≤ self.k

      Lower bound k ≥ 400md from the theorem statement.
