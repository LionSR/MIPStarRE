    Section 6 — Ordinary Self-Improvement Bridge #

    Core public API for the ordinary self-improvement package: constructors for SelfImprovementPackage, the induction-section wrapper selfImprovementInInductionSection, the monotone-witness cleanup mainInductionOfWitness, and the pasting wrapper ldPastingInInductionSection.

    The answer-valued slice constructors are separated into SelfImprovementBridge.AnswerSlice.

      Type (max u_1 u_2)

      Narrow assumption package for running the Section 9 self-improvement bridge on each Section 6 slice.

      The package deliberately keeps the remaining mathematical obligations explicit: for every slice it asks for an honest SymStrat params ι whose state, point-measurement interface, and averaged point operator agree with the restricted-slice bookkeeping used by Section 6, together with the Section 9 SelfImprovementBridgeInputs for that honest strategy. The equalities below do not derive the extra SymStrat fields (permInvState, densityFixed, or isNormalized) from the restricted strategy; those remain part of the supplied honest slice strategies.

      • sliceStrategy : Fq params → SymStrat params ι

        Honest symmetric strategies realizing the slice interfaces.

      • state_eq (x : Fq params) : (self.sliceStrategy x).state = strategy.state

        Each honest slice strategy uses the ambient state.

      • pointMeasurement_eq (x : Fq params) : (self.sliceStrategy x).pointMeasurement = (xRestrictedStrategy params strategy x).pointMeasurement

        Its point measurement agrees with the restricted-slice point interface.

      • averagedPoint_eq (x : Fq params) (h : Polynomial params) : IdxPolyFamily.averagedPointEvaluationOperator (self.sliceStrategy x) h = IdxPolyFamily.averagedSlicePointEvaluationOperator strategy x h

        Its averaged point operator is the averaged slice point operator used by Section 6.

      • good (x : Fq params) : (self.sliceStrategy x).IsGood (restrictionPkg.profile.axisParallel x) (restrictionPkg.profile.selfConsistency x) (restrictionPkg.profile.diagonal x)

        The honest slice strategy is good with the restricted failure profile.

      • bridgeInputs (x : Fq params) : SelfImprovement.SelfImprovementBridgeInputs params (self.sliceStrategy x) (restrictionPkg.profile.axisParallel x) (restrictionPkg.profile.selfConsistency x) (inductionPkg.sliceError x)

        The remaining Section 9 bridge inputs for each honest slice strategy.
