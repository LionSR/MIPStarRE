    Six-step local-variance transport-chain assembly #

    This module assembles the six steps of lem:local-variance-of-points (expansion.tex, lines 305--311) into a single triangle-inequality bound on the hypercube-edge distribution, producing the main transport estimate localVarianceTransportChainBound.

      The paper's six-step edge transport estimate in the native rerandomizeCoord presentation.

      The triangle chain is proved on the line-pair presentation above; this theorem reindexes that presentation back to the hypercube-edge sampler u, i, x ↦ (u, u[i ↦ x]), which is exactly rerandomizeCoord.
