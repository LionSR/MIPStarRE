    source
    structure MIPStarRE.LDT.GlobalVariance.MatrixVarianceTransferRealization (params : Parameters) [FieldModel params.q] :
    Type (max u_1 (u_2 + 1))

    Concrete matrix data realizing the operators and distributions used in the variance-transfer statements.

    The local measurement families live on a single prover space space, while the ambient state lives on the bipartite tensor product space ⊗ space. This matches the paper's convention that point and line operators act on the left register and polynomial weights act on the right register.

    • space : MakingMeasurementsProjective.FiniteHilbertSpace

      The local finite-dimensional Hilbert space carrying the point/line measurements.

    • state : MakingMeasurementsProjective.PositiveMatrixState (ExpansionHypercubeGraph.tensorHilbertSpace self.space self.space)

      The ambient bipartite positive matrix state.

    • pointMeasurement : Point params → MakingMeasurementsProjective.MatrixSubmeasurement (Fq params) self.space

      The point measurement family u ↦ A^u.

    • axisMeasurement : AxisParallelLine params → MakingMeasurementsProjective.MatrixSubmeasurement (DegreeBoundedLineAnswer params) self.space

      The axis-parallel line measurement family ℓ ↦ B^ℓ.

    • polynomialMeasurement : MakingMeasurementsProjective.MatrixSubmeasurement (DegreeBoundedPolynomialAnswer params) self.space

      The polynomial-weight submeasurement G.

    • axisQuestionParameter : AxisParallelLineQuestion params → Fq params

      The affine parameter used to evaluate a queried point on an axis-parallel line.
