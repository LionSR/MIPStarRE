                      noncomputable def MIPStarRE.LDT.ExpansionHypercubeGraph.matrixGlobalVarianceTraceForm (params : Parameters) (model : MatrixOperatorFamilyRealization params) :
                      Error

                      The actual trace form for the global variance.

                      The 1 / hypercubeVertexCount factor (= 1 / M where M = q^m) matches the paper's lem:global-rewrite: the global variance equals (1/M) · Tr(⟨φ⊥| ⊗ A⊥ · (I ⊗ |ψ⟩⟨ψ|) · |φ⊥⟩ ⊗ A⊥), which in turn equals (1/2) · E_{u,v} ⟨ψ| (Aᵘ − Aᵛ)² ⊗ I |ψ⟩.
