    Delta-generated topological spaces #

    This file defines the category DeltaGenerated of delta-generated spaces. This is a particular case of the construction in the file Mathlib/Topology/Convenient/Category.Lean: this is the category of X-generated spaces where X is the family of spaces Fin n → ℝ for all n : ℕ.
