    Set integral #

    In this file we prove properties of ∫ᵛ x in s, f x ∂[B; μ]. Recall that this notation is defined as ∫ᵛ x, f x ∂[B; μ.restrict s].

    The API in this file is modelled on the API for the Bochner integral.
